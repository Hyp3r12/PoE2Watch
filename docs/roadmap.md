# Roadmap

PoE2Watch is currently in the `v0.5.x-alpha` line.

## Current Roadmap

| Version | Focus | Status |
| --- | --- | --- |
| v0.4.x | Statistics, Adaptive Polling, Insights, Settings, Top Sales | Complete |
| v0.5.x | Trading Experience: Hover-style Item Cards, Better Embeds, Goals, Inventory, Autocomplete, Pagination, Statistics Export | Active |
| v0.6.x | Multi User: PostgreSQL, Multiple Guilds, User Accounts, Inviteable Bot, Sale History Search | Planned |
| v0.7.x | Website: Login, Dashboard, Public Stats, API | Planned |
| v1.0.0 | Cloud: Hosted PoE2Watch, OAuth, Managed Bot | Planned |

## v0.5.x: Trading Experience

- Hover-style item cards.
- Better Discord embeds.
- Prioritized trading goals.
- Inventory tracking.
- Slash command autocomplete.
- Pagination.
- Statistics export.

## v0.6.x: Multi User

- PostgreSQL.
- Multiple guilds.
- User accounts.
- Inviteable bot.
- Sale history search.

## v0.7.x: Website

- Login.
- Dashboard.
- Public stats.
- API.

## v1.0.0: Cloud

- Hosted PoE2Watch.
- OAuth.
- Managed bot.
- Cloud dashboard.

OAuth and cloud auth remain placeholders until official GGG guidance is available.
